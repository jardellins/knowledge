## Starting the application

Follow this steps to get start the application and enjoy play with it.

## Download

```sh

https://github.com/jardellins/knowledge.git

```

## Start

Change directory of the app

```sh

cd knowledge

```
Start with yarn

```sh
yarn install

yarn start
```
